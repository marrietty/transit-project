# Code of Conduct

## Our Pledge

We are committed to making participation in this project a positive, healthy, and respectful experience for everyone — regardless of background, experience level, or identity.

---

## Expected Behaviors

We ask everyone involved in this project to:

- Be **kind and respectful** in all interactions
- Use **welcoming and inclusive** language
- Accept and give **constructive feedback** gracefully
- Focus on what is **best for the project and community**
- Show **empathy** toward other contributors

---

## Unacceptable Behaviors

The following will not be tolerated:

- Harassment, insults, or personal attacks of any kind
- Discriminatory jokes or language
- Sharing others' private information without permission
- Any conduct that would be considered inappropriate in a professional or academic setting

---

## Enforcement

If you witness or experience behavior that violates this Code of Conduct, please report it by opening a private issue or contacting the project maintainer directly.

All reports will be reviewed and taken seriously. The project maintainer is responsible for clarifying standards and may remove, edit, or reject contributions that do not align with this Code of Conduct.

---

## Scope

This Code of Conduct applies to all project spaces — including the repository, issues, pull requests, and any other communication related to this project.
