# Contributing to TranSpec

Thank you for your interest in contributing to TranSpec! This is a student open source project, and all contributions are welcome.

---

## Before You Start

Please read the [README.md](./README.md) to understand what the project does, and review the [Code of Conduct](./CODE_OF_CONDUCT.md) before participating.

---

## How to Contribute

### Reporting a Bug

If you found something that isn't working correctly:

1. Go to the **Issues** tab in this repository
2. Click **New Issue**
3. Use this format:
   - **Title:** Short description of the bug
   - **What happened:** What you saw
   - **What you expected:** What should have happened
   - **Steps to reproduce:** How to trigger the bug

### Suggesting a Feature

Have an idea that would improve the project?

1. Open a **New Issue** and label it as a Feature Request
2. Describe what you'd like to see and why it would be useful

### Submitting Code

1. **Fork** this repository
2. **Create a new branch** for your change:
   ```bash
   git checkout -b fix/your-fix-name
   ```
3. Make your changes and **test them** locally
4. **Commit** with a clear message:
   ```bash
   git commit -m "Fix: corrected input validation for contributor name"
   ```
5. **Push** your branch and open a **Pull Request**

---

## Code Guidelines

- Keep code **readable** — use clear variable names
- Add **comments** where the logic isn't immediately obvious
- Follow the existing **style** already in `main.py`
- Do not break existing functionality when adding something new

---

## Not a Coder? You Can Still Help

- Fix **typos or grammar** in the README or documentation
- Improve **examples** or instructions
- Translate documentation to another language
- Share the project or give feedback

---

## Questions?

Open an issue and tag it with the `question` label. We'll do our best to respond.
